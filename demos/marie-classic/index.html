This website presents a proof-of-concept search engine for accessing chemical data from the World Avatar Knowledge Graph — species and thermophysical properties, zeolites, reaction mechanisms, metal–organic polyhedra, and more.

Type a question below or pick a sample question. The unified backend can also answer Singapore urban questions from this page. For advanced search and plots, use the Next.js Marie UI.

Marie is still under development; accuracy will improve with further ontology refinement.
